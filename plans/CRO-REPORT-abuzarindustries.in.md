# CRO Report — Abuzar Industries

**Primary conversion goal:** Get visitors to request a quote or contact via WhatsApp  
**Secondary goal:** Get visitors to use the timber calculator  
**Traffic:** Local organic + word-of-mouth (timber is a high-intent purchase)

---

## 1. Value Proposition Clarity — ✅ Good

"Premium timber, custom cut, delivered across Karnataka" is clear within 5 seconds. The tagline + subtext + trust bar work together well.

| Strength | Issue |
|----------|-------|
| "Chitradurga's Timber House" establishes local authority |  |
| "Three generations since 1995" builds trust |  |
| Pricing visible upfront (₹4,000, ₹2,800, ₹1,000) |  |

## 2. Headline — 🟡 Could convert harder

Current: *"Trusted Timber Specialists"* → already updated to *"Premium Timber, Cut Your Way"* (stronger)

## 3. CTA — 🔴 Main issue

| CTA | Strength | Weakness |
|-----|----------|----------|
| **Build a Quotation** (primary) | Strong action verb | Goes to calculator page (funnel drop risk) |
| **WhatsApp Us** (secondary) | Low friction | Generic prefill — no dimension context |

The biggest issue: two CTAs of equal visual weight. When everything is primary, nothing is.

## 4. Trust Signals — ✅ Good

Trust bar (Trusted Since 1995, Across Karnataka, Custom Cut Sizes) + WhatsApp social proof

## 5. Friction Points — 🟡

- Calculator requires manual dimension entry before seeing price
- No testimonials or reviews visible on homepage
- No photos of finished projects or happy customers

---

## Quick Wins (Implement Now)

### 1. Make "Build a Quotation" the clear primary CTA
The WhatsApp button should visually recede — make it an outline/tone-down variant so there's one hero action.

### 2. Add WhatsApp prefill with dimension context
When user clicks WhatsApp from the calculator, include the actual dimensions typed out. Already done in QuoteTool.

### 3. Add testimonial snippet to the trust bar area
Even one quote from a builder would massively boost conversion. Currently zero social proof beyond the "since 1995" badge.

---

## High-Impact Changes

### 1. Add 2-3 customer logos or testimonials to the homepage
Even text-only quotes like *"Abuzar supplies all our teak — quality is consistently excellent"* would convert cold visitors.

### 2. Show live WhatsApp response time
Replace "Quick replies within minutes" with something concrete: "Usually replies within 5 min" or add a snippet from actual WhatsApp chat.

### 3. Add project photos to the gallery/product section
People buy timber by seeing what it looks like in use. Add a "Projects" section with finished doors, furniture, construction.

### 4. Calculator: add a "Send Quote via WhatsApp" button alongside the estimate
Currently user must manually copy and switch apps. One-click send would dramatically increase quote completions.

---

## Test Ideas

- **CTA color test**: ochre (gold) CTA vs walnut (brown) CTA — which drives more clicks
- **WhatsApp prefill test**: does pre-filling dimensions increase quote completion rate?
- **Price display test**: showing ₹4,000/cu.ft upfront vs "Call for price" — transparency increases trust
